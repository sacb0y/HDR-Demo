# HDR Demo
HDR Learning tool for developers, enthusiasts and first timers

## For Developers
For developers this project being on github provides a few resources currently
- Display Info script includes examples of how to read a users display capabilities
- HDR Gallery of reference HDR games
- SDR vs HDR comparisons to understand the benefits of HDR implementation (still early)
- UI navigation flow is handled with a node set up made with Graph Toolkit, so if you've ever wondered how that could work there you go.
  
Planned for the future:
- Gameplay demo's showing how HDR can influence design
- Tonemapping examples to understand how tonemapping can affect your game
- Tonemapping tools (additional tonemapping postprocessing scripts to compare with)

## For Enthusiasts:
For HDR enthusiasts this provides easy access to display testing
- Local Dimming, ABL, Color Banding, and peak brightness tests included
- HDR Gallery for quick previewing and comparison
- Works on ultra-wide displays

Planned for the Future:
- Additional tests for determining display specs, and tools for motion clarity
- playable demo for real-time testing

## For First-timers new to HDR
- Test's include explanations for their use and purpose
- Explanations for what HDR is with examples
- Toggle HDR on and off in any test to see the difference in realtime.
  
Planned for the Future:
- Playable demos showing how HDR benefits games
- Guidence for shopping for HDR displays to avoid scam displays
